# Junye Zhou — portfolio site

Static portfolio hosted on **GitHub Pages**.

## Live URL

After deployment: **https://moonlitjaderabbit.github.io/website/**

## Local preview

Open `index.html` in a browser, or:

```bash
npx --yes serve .
```

## Deploy

Pushes to `main` run `.github/workflows/pages.yml` and update the live site.
